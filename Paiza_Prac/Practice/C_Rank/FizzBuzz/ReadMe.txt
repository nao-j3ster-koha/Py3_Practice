【課題概要】
整数 N が入力として与えられる。

1からNまでの数値を昇順に表示させる。

ただし、表示する数値が

・3の倍数、かつ5の倍数のとき ："Fizz Buzz"
・3の倍数のとき            ："Fizz"
・5の倍数のとき            ："Buzz"

のように表示させる。

ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

【入力される値とフォーマット】
正の整数 N



【期待する出力】
各数値を１つずつ表示させて、改行していく
余計な文字、空行は含まない。


【条件】
数値は以下の条件を満たす。

1 <= N <= 100
