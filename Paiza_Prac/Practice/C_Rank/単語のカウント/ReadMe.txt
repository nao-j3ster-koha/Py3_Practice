【課題概要】
半角スペースで区切られた長さNの文字列

【期待される出力】
単語、半角スペース、出現回数の順で１行に１単語で出現したすべての単語を、列に出現する順に出力してください

【条件】
1 <= N <= 1000


【入力・出力例】
（例１）
＜入力＞

red green blue blue green blue

＜出力＞
red 1
green 2
blue 3


（例２）
＜入力＞
Nagato Yukikaze Akagi Kitakami Nagato Akagi Akagi Kitakami

＜出力＞
Nagato 2
Yukikaze 1
Akagi 3
Kitakami 2
