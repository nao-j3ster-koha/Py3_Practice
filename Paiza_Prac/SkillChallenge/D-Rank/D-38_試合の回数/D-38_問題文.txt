【問題】

あなたは1対1で戦う競技の大会を開くことにしました。
N 人の選手で総当り対戦を行う場合、何回の対戦が行われるか出力して下さい。
N 人の総当り戦では N 人がそれぞれ自分自身以外と対戦します。
また、自分自身から見た結果と、相手側から見た結果があるので N × (N - 1) / 2で求められます。

【入力】
入力は以下のフォーマットで与えられます。

N
N は大会に参加する選手の数

入力値最終行の末尾に改行が１つ入ります。

【出力】
N 人の選手で総当り対戦を行う場合、何回の対戦が行われるか出力して下さい。
最後は改行し、余計な文字、空行を含んではいけません。

【条件】
すべてのテストケースで以下の条件を満たします。
入力される値は全て整数

・2 ≦ N ≦ 100
